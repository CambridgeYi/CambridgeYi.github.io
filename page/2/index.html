linux常用命令

  1. 修改文件所属的组

    ==chgrp -R 用户组名 文件名==

  2. 修改文件所有者

    chown -R 用户名 文件名

    ==-R ****表示递归目录及子目录****==

  3. linu下添加，删除，修改，查看用户和用户组

    gpasswd -a user1 group1 ==将用户user1添加到group1组中（用户可以属于多个组）==

  4. mkdir -p

    ==-p p表示parent 自动创建多级目录==

  5. hostnamectl –static set-hostname ==++nname++==

    ==设置主机机器名==

  6. sudo netstat -anp |grep PID或port

    ==通过进程PID查看端口占用或通过port查看PID==

  7. ps -ef |grep port或PID

    ==查看进程PID或port启动情况==
