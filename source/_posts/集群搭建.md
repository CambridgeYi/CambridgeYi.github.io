#### 命令
```
# mysql创建账号
CREATE USER  'slave'@'%' IDENTIFIED WITH mysql_native_password BY '123456'; 
# mysql授予用户 slave REPLICATION SLAVE权限和REPLICATION CLIENT权限
GRANT REPLICATION SLAVE, REPLICATION CLIENT ON *.* TO 'slave'@'%';
```
