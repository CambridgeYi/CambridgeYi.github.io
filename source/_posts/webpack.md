#### npm初始化项目
```
npm init -f # -f表示force的意思，不加这个，npm会让你输入一堆信息，比如name、version之类，如果只是做做实验小demo，直接-f，npm帮你初始化package.json，并填充各种默认值，省事
```

#### vue初始化webpack项目
```
vue init webpack 项目名称
```
#### npm安装
```
npm i 包名称 # i表示安装到当前目录
```
